
\subsection{Basic examples}

\begin{proposition}
The following sets are all \'etale sheaves: 
\begin{itemize}
\item Any scheme is an \'etale sheaf.
\item Any finitely presented modules is an \'etale sheaf.
\item Any finitely presented algebra is an \'etale sheaf.
\end{itemize}
\end{proposition}

\begin{proof}
See Severi-Brauer draft. Affine schemes, f.p. algebra and f.p. and co-f.p. module immediately follow from $R$ being and \'etale sheaf and duality. For scheme we crucially use \Cref{etale-characterisation-monic}. 
\end{proof}
